# Banking Application

This is a **web-based banking application** inspired by the design of the CIBC banking system. The application allows users to create an account, securely log in, and transfer money between users.

The project is built using **React, TypeScript, JavaScript, HTML, Node.js, and MongoDB**, following a full-stack development approach. It focuses on implementing core banking features such as authentication and money transfer while demonstrating modern web development practices.

This application is developed for learning and portfolio purposes to showcase full-stack development skills.
